## Prerequisites

- [NodeJs](https://nodejs.org/en/download/)

## Installing

```bash

  # Downloading this repositories
  $ git clone https://github.com/pereiradilson/nivel-01-frontend-desafio.git

  # Enter folder
  $ cd nivel-01-frontend-desafio
  
  # Install repositories
  $ yarn
  
  # Run project
  $ yarn start
  
```

## Running the tests

```bash

  # Execute tests
  $ yarn test
  
```
