# GoSatck - Challenge: ReactJS concepts

Creating a front-end with ReactJS to list, register and remove repositories.

## Prerequisites

- [NodeJs](https://nodejs.org/en/download/)

## Installing

```
yarn
```
or
```
npm install
```

## Running the tests

```
yarn test
```
or
```
npm run test
```
